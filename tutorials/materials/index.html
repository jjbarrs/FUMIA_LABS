Instalación de Material

Los proyectos de esta clase utilizan Python 3. Este tutorial cubre:

  • Instrucciones sobre cómo configurar Python
  • Un tutorial de Python mini

Instalación de Python

Necesitan una distribución de Python (3.9 - 3.11) y ya sea pip o conda.

Para verificar si cumplen con nuestros requisitos, deben abrir la terminal y ejecutar python -V y ver que la versión sea al menos 3.9. Luego, ejecuten pip -V o conda -V y vean que al menos uno de estos funcione e imprima alguna versión de la herramienta. En algunos sistemas que también tienen Python 2, es posible que deban usar python3 y pip3 en lugar de los mencionados anteriormente.

Hay dos formas de instalar Python para este curso, la primera es descargarlo directamente desde el sitio web oficial. La segunda, que recomendamos encarecidamente, es configurar un entorno conda.

Configurando un entorno Conda (Recomendado)

Para evitar tener dependencias conflictivas entre esta clase y cualquier otro proyecto en el que trabaje, recomendamos configurar un entorno virtual para este curso. Esto facilita la instalación de dependencias y evita la mayoría de los problemas comunes de instalación.

  1. Instalar Miniconda: Miniconda es una herramienta ligera que le permite crear entornos virtuales, así como le da acceso a pip y algunos paquetes esenciales de Python.
  2. Corran conda create -n [nombre de su entorno] python=3.11 en su terminal/Anaconda Prompt
  3. Activen el entorno con conda activate [nombre de su entorno]. Ahora que están en el entorno, cualquier dependencia que instalen solo será accesible dentro de su entorno virtual.

Ahora que tienen un entorno virtual configurado, asegúrense de activarlo antes de instalar cualquier dependencia o ejecutar su proyecto. Si alguna vez olvidan el nombre de su entorno, ejecuten conda env list para obtener una lista completa de los entornos que han creado.

Descargando Python directamente

Nota: Este método es más probable que cause problemas de dependencia, y también es más difícil resolver esos problemas cuando surgen.

  1. Instalar Python 3.11:
    • Para Windows y MacOS, recomendamos usar un instalador gráfico oficial: descarguen e instalen Python 3.11.
    • Para Linux, usen un administrador de paquetes para instalar Python 3.11.
  2. Instalar Pip:
    • Para Windows y MacOS, ejecuten python -m ensurepip --upgrade.
    • Para Linux, usen un administrador de paquetes para instalar Pip para Python 3.

Instalación de Dependencias

Algunos proyectos tienen dependencias adicionales. Es importante instalarlas ahora para que si hay un problema con la instalación de Python, no tengamos que volver o volver a hacer la instalación más tarde.

Las librerías que necesitarán para este curso son:

  • numpy: Una librería de Python que proporciona soporte para arreglos y matrices multidimensionales, junto con una colección de funciones matemáticas de alto nivel para operar en estos arreglos.
  • matplotlib: Una librería de Python para trazar gráficos 2D.

Para instalar estas dependencias, ejecuten:

pip install numpy
pip install matplotlib

Para comprobar que todo está funcionando, ingresen al intérprete de Python:

python

y ejecuten:

import numpy as np
import matplotlib.pyplot as plt

Solución de Problemas

Algunas instalaciones tendrán python3 y pip3 refiriéndose a lo que queremos usar. Además, puede haber múltiples instalaciones de python que pueden complicar qué comandos instalan dónde.

Pueden verificar la instalación de Python y pip con los siguientes comandos:

python -V  # version de python
pip -V     # version de pip y para que version de python esta instalando
which python  # ubicacion de python

Si hay un error de importación de tkinter, es probable que Python sea atípico y provenga de homebrew. Desinstalen eso e instalen Python desde homebrew con soporte para tkinter, o usen el instalador gráfico recomendado.

Si necesitan ayuda con la depuración, intenten configurar un entorno conda antes de comunicarse a través de Slack, ya que hacer esto soluciona la mayoría de los errores de instalación.

Tutorial de Python

El objetivo de este tutorial es que ustedes puedan aprender sobre lo básico de Python 3, no esperamos que se vuelvan expertos con esta introducción. En este curso no es fundamental que dominen al 100% este lenguaje de programación y realmente solo necesitan conocer lo básico para poder completar satisfactoriamente los proyectos.

Introducción

Python es un lenguaje de programación multiparadigma y con tipos dinámicos. El código escrito en Python se dice que es similar al pseudocódigo, ya que permite expresar ideas bastante poderosas en muy pocas líneas de código y al mismo tiempo ser bastante leíble. Como ejemplo, aquí hay una implementación del clásico algoritmo quicksort escrito en Python:

def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

print(quicksort([3,6,8,10,1,2,1]))

Tipos de datos básicos

Como en muchos lenguajes, Python tiene un número de tipos de datos básicos incluyendo enteros, floats, booleans y strings. Estos tipos de datos se comportan como lo han visto en otros lenguajes de programación.

Números

Los enteros y los floats funcionan como ustedes han visto que funcionan en otros lenguajes:

x = 3
print(type(x))  # type: sirve para obtener el tipo de un objeto
print(x)
print(x + 1)
print(x - 1)
print(x * 2)
print(x / 2)    # division exacta
print(x // 2)   # division entera
print(x ** 3)   # exponentes
x += 1
print(x)
x *= 2
print(x)
y = 2.5
print(type(y))
print(y)
print(y + 1)
print(y * 2)
print(y / 2)
print(y ** 2)

Noten que Python no tiene los operadores de incremento unario (x++) o decremento (x—). Python también tiene tipos para manejo de números complejos, todo esto lo pueden ver en la documentación oficial.

Booleans

Python implementa todos los operadores usuales para hacer lógica booleana, pero usa palabras en Inglés en vez de símbolos (&&, ||, etc.):

t = True
f = False
print(type(t))
print(t and f)  # AND lógico
print(t or f)   # OR lógico
print(not t)    # NOT lógico
print(t != f)   # XOR lógico

Strings

Python tiene buen soporte de strings:

hello = 'hello'  # las literales string pueden ser declaradas con apóstofre
world = "world"  # o comillas, no importa la verdad...
print(hello)
print(len(hello))  # con len obtenemos el length del string
hw = hello + ' ' + world  # concatenación, así como en Java
print(hw)
hw12 = '%s %s %d' % (hello, world, 12)  # formato de texto al estilo sprintf
print(hw12)

Los objetos de tipo string tienen un montón de métodos útiles; Por ejemplo:

s = 'hello'
# Pone la letra del principio en mayuscula
print(s.capitalize())
# Pasa el string a mayusculas
print(s.upper())
# Justifica un string, rellenando con espacios
print(s.rjust(7))
# Centra un string, rellanando con espacios
print(s.center(7))
# Reemplaza todas las apariciones de un substring por otro
print(s.replace('l', '(ell)'))
# Quita todos los caracteres de whitespace del principio y final
print('  world '.strip())

Pueden encontrar una lista de todos los métodos en la documentación.

Contenedores

Python incluye bastantes tipos de contenedores: listas, diccionarios, sets y tuplas.

Listas

Una lista en Python es equivalente a un arreglo en C o Java, pero es dinámico (puede cambiar su tamaño, es decir no es fijo) y puede tener elementos de diferentes tipos:

# Crea una lista
xs = [3, 1, 2]
print(xs)
# Para acceder a un elemento de la lista, los indices van de 0 a length - 1
print(xs[1])
# Los indices negativos empiezan desde el final de la lista
print(xs[-1])
# Las listas pueden contener elementos de diferentes tipos
xs[2] = 'foo'
print(xs)
# Añade un elemento al final de la lista
xs.append('bar')
print(xs)
# Quita y devuelve el último elemento de la lista
x = xs.pop()
print(x, xs)
# para obtener el length de la lista
print(len(xs))

Pueden encontrar más detalles en la documentación de listas.

Slicing

Además de acceder a los elementos de una lista uno a la vez, Python provee una sintaxis para acceder a sublistas, a esto se le conoce como slicing:

# Creamos una nueva lista
nums = [0, 1, 2, 3, 4]
print(nums)
# Agarra una sublista desde el indice 2 hasta el indice 3 es decir [2,4)
print(nums[2:4])
# Agarra una sublista desde el indice 2 hasta el final de la lista, es decir [2,4]
print(nums[2:])
# Agarra una sublista desde el principio de la misma hasta el indice 2 sin incluir, es decir [0,2)
print(nums[:2])
# Agarra toda la lista "[0, 1, 2, 3, 4]"
print(nums[:])
# El slice puede ser negativo, en este caso agarra desde el principio hasta el penultimo elemento, es decir [0,3]
print(nums[:-1])
# Asigna una sublista al slice
nums[2:4] = [8, 9]
print(nums)
Ciclos

Pueden iterar sobre los elementos de una lista:

animals = ['cat', 'dog', 'monkey']
for animal in animals:
    print(animal)

Si quieren acceso al índice de cada elemento en el body del ciclo, usen la función enumerate:

animals = ['cat', 'dog', 'monkey']
for idx, animal in enumerate(animals):
    print('#%d: %s' % (idx + 1, animal))
Listas por comprensión

Al programar, seguido queremos transformar un tipo de dato en otro. Como un ejemplo simple, consideren el siguiente código que computa números al cuadrado:

nums = [0, 1, 2, 3, 4]
squares = []
for x in nums:
    squares.append(x ** 2)
print(squares)

Pueden hacer esto más simple usando una list comprehension:

nums = [0, 1, 2, 3, 4]
squares = [x ** 2 for x in nums]
print(squares)

Las list comprehensions también pueden tener condiciones:

nums = [0, 1, 2, 3, 4]
even_squares = [x ** 2 for x in nums if x % 2 == 0]
print(even_squares)

Diccionarios

Un diccionario guarda pares (llave, valor), similar a un Map en Java o un objeto en Javascript:

d = {'cat': 'cute', 'dog': 'furry'}  # Crea un nuevo diccionario con algunos datos
print(d['cat'])       # Obtén el valor de una llave; imprime "cute"
print('cat' in d)     # Checa si el diccionario tiene una llave dada; imprime "True"
d['fish'] = 'wet'     # Asigna un valor a una llave
print(d['fish'])      # Imprime "wet"
# print(d['monkey'])  # KeyError: 'monkey' no es una llave de d
print(d.get('monkey', 'N/A'))  # Obtén el valor con un valor default; imprime "N/A"
print(d.get('fish', 'N/A'))    # Obtén el valor con un valor default; imprime "wet"
del d['fish']         # Quita un elemento del diccionario
print(d.get('fish', 'N/A')) # "fish" ya no es una llave; imprime "N/A"

Pueden encontrar todo lo que necesiten saber sobre diccionarios en la documentación.

Ciclos

Es fácil iterar sobre las llaves en un diccionario:

d = {'person': 2, 'cat': 4, 'spider': 8}
for animal in d:
    legs = d[animal]
    print('A %s has %d legs' % (animal, legs))

Si quieren acceso a las llaves y sus valores correspondientes, usen el método items:

d = {'person': 2, 'cat': 4, 'spider': 8}
for animal, legs in d.items():
    print('A %s has %d legs' % (animal, legs))
Diccionarios por comprensión

Estos son similares a list comprehensions, pero les permiten construir diccionarios fácilmente:

nums = [0, 1, 2, 3, 4]
even_num_to_square = {x: x ** 2 for x in nums if x % 2 == 0}
print(even_num_to_square)

Sets

Un set es una colección no ordenada de elementos distintos:

animals = {'cat', 'dog'}
print('cat' in animals)   # Checa si un elemento está en un set; imprime "True"
print('fish' in animals)  # imprime "False"
animals.add('fish')       # Agrega un elemento a un set
print('fish' in animals)  # imprime "True"
print(len(animals))       # Número de elementos en un set; imprime "3"
animals.add('cat')        # Agregar un elemento que ya está en el set no hace nada
print(len(animals))       # imprime "3"
animals.remove('cat')     # Quita un elemento de un set
print(len(animals))       # imprime "2"
Ciclos

Iterar sobre un set tiene la misma sintaxis que iterar sobre una lista; sin embargo ya que los sets son no ordenados, no pueden hacer suposiciones sobre el orden en el que visitan los elementos del set:

animals = {'cat', 'dog', 'fish'}
for idx, animal in enumerate(animals):
    print('#%d: %s' % (idx + 1, animal))
Set comprehensions

De manera similar a las listas y los diccionarios, podemos construir sets usando set comprehensions:

from math import sqrt
nums = {int(sqrt(x)) for x in range(30)}
print(nums)

Tuplas

Una tupla es una lista inmutable (ordenada) de valores. Una tupla es en muchas formas similar a una lista; una de las diferencias más importantes es que las tuplas pueden ser utilizadas como llaves en diccionarios y como elementos de sets, mientras que las listas no pueden. Aquí hay un ejemplo trivial:

d = {(x, x + 1): x for x in range(10)}  # Crea un diccionario con llaves de tupla
t = (5, 6)        # Crea una tupla
print(type(t))    # imprime "<class 'tuple'>"
print(d[t])       # imprime "5"
print(d[(1, 2)])  # imprime "1"

Pueden leer más sobre tuplas en la documentación.

Funciones

Las funciones en Python son definidas usando la keyword def:

def sign(x):
    if x > 0:
        return 'positive'
    elif x < 0:
        return 'negative'
    else:
        return 'zero'

for x in [-1, 0, 1]:
    print(sign(x))

Frecuentemente definimos funciones que toman argumentos opcionales keyword, así:

def hello(name, loud=False):
    if loud:
        print('HELLO, %s!' % name.upper())
    else:
        print('Hello, %s' % name)

hello('Bob')  # Imprime "Hello, Bob"
hello('Fred', loud=True)  # Imprime "HELLO, FRED!"

Hay mucho más de que hablar sobre funciones de Python en la documentación.

Funciones Lambda

Python soporta la creación de funciones anónimas (es decir funciones que no están enlazadas a un nombre) en tiempo de ejecución, usando un constructor llamado lambda. Usualmente esto no se usa para funciones complejas, sino para funciones pequeñas:

# Función lambda que retorna la suma de sus argumentos
f = lambda x, y: x + y
print(f(1, 2))  # Imprime "3"

Clases

La sintaxis para definir clases en Python es sencilla:

class Greeter(object):
    # Constructor
    def __init__(self, name):
        self.name = name  # Crea una variable de instancia

    # Método de instancia
    def greet(self, loud=False):
        if loud:
            print('HELLO, %s!' % self.name.upper())
        else:
            print('Hello, %s' % self.name)

g = Greeter('Fred')  # Construye una instancia de la clase Greeter
g.greet()            # Llama un método de instancia; imprime "Hello, Fred"
g.greet(loud=True)   # Llama un método de instancia; imprime "HELLO, FRED!"

Pueden leer mucho más sobre clases en Python en la documentación.
